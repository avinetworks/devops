Announcing NSX Advanced Load Balancer (Avi) with Cloud Services. This is a significant step toward simplifying application delivery and enabling customers to easily consume application services. Avi with Cloud Services is now available and provides a consistent set of application services including local and global load balancing, web application firewall (WAF), and container ingress. It also provides a set of operational services (aka PULSE) including central licensing services, live security threat updates, proactive zero-touch support, and more.

Customers now have flexibility to consume Avi as they may choose to manage their own Avi Controllers and deploy Avi Service Engines closest to the applications or get hosted controller services from VMware.

Learn more about Avi with Cloud Services and how to: 

  • Simplify your operational experience for application delivery through secure and one-click cloud services
  • Enable seamless global capacity management for DR and cloud bursting use cases
  • See a live demo of real-time application protection through live security threat intelligence
